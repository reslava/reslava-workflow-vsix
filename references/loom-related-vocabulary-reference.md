### 🧵 Core Loom/Weaving Words

| English | Spanish | Vibe |
|---------|---------|------|
| Loom | Telar | The tool itself |
| Weave | Tejer | The action |
| Thread | Hilo | Single unit of work |
| Yarn | Hilo / Estambre | Raw material, ideas |
| Fabric | Tejido | The final result |
| Tapestry | Tapiz | Complex, beautiful result |
| Pattern | Patrón | Structure, design |
| Stitch | Puntada | Small precise action |
| Knot | Nudo | Connection point |
| Spindle | Huso | Core element |
